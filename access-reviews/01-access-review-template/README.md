## Purpose
This folder contains the standardized template used to perform identity and access reviews in a repeatable, auditable, and defensible manner.

The goal of this template is to ensure users have the right access for their role, prevent permission creep, and produce clear evidence suitable for audits.

## When to Use
- Periodic access reviews
- Privileged access reviews
- Joiner / mover / leaver follow-ups
- Audit or investigation support

## Ownership
- Business: attests access
- IAM: validates and remediates
- Security / Compliance: oversight as required
