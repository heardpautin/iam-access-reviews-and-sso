# Sample Access Review Data

## Purpose
This folder contains sample, non-production data used to demonstrate
the structure of access review inputs and outputs.

The data is intentionally fictional and does not represent any real
organization, system, or user.

## Contents
- `sample-access-review.csv`
  - Simulated export of user access and entitlements
  - Used for access review walkthroughs, validation, and analysis

## Usage
This data may be used to:
- Practice access review analysis
- Identify over-privileged access
- Validate role alignment
- Demonstrate audit-ready review artifacts

## Data Safety
- No real users
- No real applications
- No sensitive or regulated data
